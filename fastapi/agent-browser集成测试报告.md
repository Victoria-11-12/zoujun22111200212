## 遇到的问题及解决方案

### 1. 编码问题
**问题描述**：Windows 控制台 GBK 编码与 Unicode 字符冲突，导致特殊符号显示乱码
**解决方案**：
- 替换所有 Unicode 符号为文本标签（[OK]/[FAIL]/[WARN]）
- 使用 UTF-8 解码处理命令行输出
- 实现安全的打印函数避免编码错误

### 2. 命令执行问题
**问题描述**：Windows PowerShell 中 "&&" 分隔符不兼容
**解决方案**：
- 使用 ";" 作为命令分隔符
- 统一使用 shell=True 参数执行命令

### 3. 元素定位问题（已解决）
**问题描述**：百度百科页面搜索框无法通过常见CSS选择器定位
**尝试的选择器**：
- `input[name="word"]`
- `#query`
- `input[placeholder*="百科"]`
- `input[type="text"]`

**最终解决方案**：
- 使用 `agent-browser snapshot -i` 命令获取页面交互元素快照
- 通过快照发现搜索框的引用标识：`textbox [ref=e84]`
- 使用 `agent-browser fill @e84 "战狼2"` 填充搜索框
- 使用 `agent-browser click @e71` 点击"进入词条"按钮
- 成功实现搜索功能，跳转到正确的词条页面
