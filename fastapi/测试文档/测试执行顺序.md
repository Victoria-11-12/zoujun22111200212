# 测试执行顺序

## 第一阶段：单元测试

### 1. 数据模型测试
```bash
pytest tests/unit/models/
```

### 2. 历史管理测试
```bash
pytest tests/unit/constants/test_history.py
```

### 3. SQL安全检测测试
```bash
pytest tests/unit/tools/test_admin_tools.py::TestDangerousKeywords
pytest tests/unit/tools/test_admin_tools.py::TestCheckSqlSafety
```

### 4. 工作流纯函数测试
```bash
pytest tests/unit/workflows/test_chart_workflow.py
```

### 5. 常量测试
```bash
pytest tests/unit/constants/
```

### 6. 工具函数测试
```bash
pytest tests/unit/tools/
```

### 7. 服务层单元测试
```bash
pytest tests/unit/services/
```

---

## 第二阶段：集成测试

### 1. 链层集成测试
```bash
pytest tests/integration/chains/
```

### 2. Agent集成测试
```bash
pytest tests/integration/agents/
```

### 3. 工作流集成测试
```bash
pytest tests/integration/workflows/
```

### 4. 服务层集成测试
```bash
pytest tests/integration/services/
```

---

## 第三阶段：端到端测试

### 1. 用户接口测试
```bash
pytest tests/e2e/test_user_router.py
```

### 2. 管理员接口测试
```bash
pytest tests/e2e/test_admin_router.py
```

### 3. 图表接口测试
```bash
pytest tests/e2e/test_chart_router.py
```

### 4. 评估接口测试
```bash
pytest tests/e2e/test_analyst_router.py
```
