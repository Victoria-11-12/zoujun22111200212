# 端到端测试 (E2E Tests)

## 3.1 用户对话接口

**文件**: `app/routers/user.py`

| 测试对象 | 测试内容 | 测试类型 |
|---------|---------|---------|
| POST /api/ai/stream | 完整对话流程、流式响应格式 | E2E测试 |

**场景覆盖**:
- 正常对话请求 → 直接回复
- 需要SQL查询的请求 → 数据库查询 → 结果包装回复
- 数据库无结果时浏览器搜索 → SQL查询无数据 → Web搜索 → 结果返回
- 安全威胁请求 → 警告拦截 → 安全日志记录
- 多轮对话 → 历史上下文保持
- 异常处理 → 错误流式返回

## 3.3 图表生成接口

**文件**: `app/routers/chart.py`

| 测试对象 | 测试内容 | 测试类型 |
|---------|---------|---------|
| POST /api/chart/generate | 完整图表生成流程 | E2E测试 |

**场景覆盖**:

- 沙箱执行失败 → 错误处理 → 日志记录
- 安全代码检测 → 危险代码拦截
