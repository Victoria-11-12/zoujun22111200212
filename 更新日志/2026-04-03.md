## 技术细节

### 对数转换原理
```python
# 训练时转换
y_train_log = np.log1p(y_train)
model.fit(X_train, y_train_log)

# 预测时逆转换
y_pred_log = model.predict(X_test)
y_pred = np.expm1(y_pred_log)  # 转换回原始尺度
```

### 过拟合控制
- 通过增加 `min_samples_split` 和 `min_samples_leaf` 限制树的生长
- 使用 `max_features='sqrt'` 限制每棵树使用的特征数
- 训练集与测试集 R² 差距控制在合理范围（约 0.2）
